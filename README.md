# YES-Project

Computer Parts Online Store

This is an e-commerce website for selling computer hardware and accessories such as CPUs, GPUs, motherboards, RAM, storage devices, and peripherals.

Features
	•	Product listing with details and prices
	•	Search and category filtering
	•	Shopping cart and checkout system
	•	User-friendly interface

Purpose

The goal of this project is to provide an easy and reliable platform for users to browse and purchase computer components online.
